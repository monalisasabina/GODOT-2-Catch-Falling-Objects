# Part 5: Types of Nodes (Main Categories)

## 1. 2D Nodes(2D Games)

### Node2D
- Basic 2D position node
- Has position (x, y), rotation, scale
- Used as a container/parent for other 2D nodes

### Sprite2D
- Displays a 2D image/texture
- Your coins, player, enemies all use this
- Can flip, rotate, scale, tint the image
